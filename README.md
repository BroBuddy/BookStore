# BookStore
A little angularJS web application with a smooth nodeJS backend.

## Tools
- angularJS
- body-parser
- bootstrap
- bower
- express
- git
- jquery
- mongodb
- mongoose
- nodemon
- nodeJS

## Installation
- git clone [project]
- npm install
- bower install
- run "node app" | "nodemon"
- run "mongod" | "mongo"
